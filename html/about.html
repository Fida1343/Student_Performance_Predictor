About This System

ML-Based System for Predicting College Student Academic Performance

Project Overview

A web-based system demonstrating machine learning principles for predicting student academic performance using a Flask backend and a Random Forest regression model.

Features

  • ML-powered prediction using Random Forest Regressor
  • Backend API built with Flask
  • Real-time results with visual analytics
  • 12 input features across academic and behavioral domains
  • Fully responsive design

How It Works

  1. Enter your student details across two sections
  2. Review and submit for ML analysis
  3. Receive predicted end-semester marks and grade

Input Features

📚 Academic (8)

  • Attendance: Percentage of classes attended (0–100%)
  • Study Hours: Daily self-study hours (0–12)
  • Previous CGPA: Last semester CGPA (0–10)
  • Internal Marks: Internal assessment score (0–30)
  • Mid-Semester Marks: Midsem exam score (0–20)
  • Backlogs: Number of failed subjects (0–5+)
  • Assignment Completion: Submission rate (0–100%)
  • Class Participation: Engagement level (1–5)

👤 Demographic & Behavioral (4)

  • Age: Student age (17–30)
  • Gender: Male / Female / Other
  • Extra-Curricular: Participation in non-academic activities
  • Late-Night Entry: Hostel entry frequency (1–5)

Output & Grading

ML Output

Predicted End-Semester Marks: 0–50

Total Marks

Total = Mid-Semester (20) + Internal (30) + End-Semester (50) = 100

Grade Scale

O 90–100  Outstanding
E 80–89  Excellent
A 70–79  Very Good
B 60–69  Good
C 50–59  Average
D 40–49  Below Average
F <40  Fail

Grade is computed using fixed academic rules, not predicted by ML.
